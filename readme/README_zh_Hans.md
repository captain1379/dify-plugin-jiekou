# 概述

Jiekou.AI（接口 AI）是一个一站式平台，为企业提供可靠、高性能的大语言模型 API 服务。我们的解决方案支持多种 AI 模态——包括自然语言、视觉、音频和视频——并汇集了 OpenAI、Anthropic、Gemini、DeepSeek、Qwen 等领先的创新成果。通过 Jiekou.AI，开发者可以轻松利用 AI 技术的最新进展，无需深厚的机器学习背景。

# 配置

1. 插件安装完成后，进入【设置】页面输入您的 API Key。

![](_assets/jiekou-01.png)

2. [注册 Jiekou.AI 账户。](https://jiekou.ai/user/login?utm_source=github_dify)
3. 在[此处](https://jiekou.ai/settings/key-management)生成并保存您的 API Key。
   - 点击【用户头像】→【API Key 管理】进入控制台。
     ![](_assets/jiekou-02.png)
   - 点击 **【+ 创建】** 生成新的 API Key。自定义一个名称，请注意密钥**仅在创建时显示一次**——请务必复制并妥善保存以便后续使用。
     ![](_assets/jiekou-03.png)
